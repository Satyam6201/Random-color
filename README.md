# 🎨 Random Color Generator

A simple and modern **Random Color Generator** built with **HTML, CSS, and JavaScript**. It allows users to generate random RGB colors, copy them to the clipboard, toggle between light/dark mode, view color history, and enjoy a smooth interactive UI experience. 🚀

---

## ✨ Features

- 🎨 Random Color Generation (RGB)
- 📋 Copy Color to Clipboard
- 🌗 Toggle Light/Dark Mode
- 🕘 Color History (up to 5 recent colors)
- 🔁 Animated Transitions
- 📱 Fully Responsive Design
- 💾 Save Favorite Colors (local storage)
- 🌈 Hover effects on previous colors
- ✅ Alert messages & animations
- 🦾 Accessibility-friendly
